# Rust Life

This is an experiment to get better at Rust programming in general, and
also to learn more about [Ratatui](https://ratatui.rs/).

![Rust Life screenshot](img/screenshot.jpg)

## Controls

Roguelike cursor keys:

```
y k u
 \|/
h-+-l
 /|\
b j n
```

```
s: Step
t: Toggle cell
q: Quit
```
