# Rust Life

This is an experiment to get better at Rust programming in general, and
also to learn more about [Ratatui].

## Controls

BSD-robots cursor keys:

```
y k u
 \|/
h-+-l
 /|\
b j n
```

```
s: Step
q: Quit
```
